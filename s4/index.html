Block 4. Architectural Frame

Architecture Cannot Be Improvised

Chaos
From improvisation to constraints

If architecture is not defined, the agent will invent it (usually wrong). Architecture defines the solution space.

Bot & Backend Are Separate

  • • The Telegram bot is the interface layer.
  • • The backend is the layer of logic, data, and integrations.
  • • The bot must not become the whole application.
  • • Separation makes the system mature and extensible.

Direct DB access destroys the boundary of responsibility. Business logic leaks into the bot layer.

Rule: Bot Talks Only to Backend API

  • • The bot should know only the backend API contract.
  • • It should not know internal models, tables, or repositories.
  • • API creates a clean boundary between interface and system.
  • • Gives consistency, replaceability, and testability.
