# Prompt: Article Summary (F02)

Variables: `{{title}}`, `{{content}}`.

```
Ты - профессиональный автор статей. Переведи название на русский: """{{title}}""", оставь только переведенное название лаконичное и выдели его жирным на примере так с одной звездочкой:
*Выпускники медицинских вузов активно выбирают психиатрию несмотря на рост ИИ-приложений в сфере ментального здоровья*

и далее через пустую строку Суммаризируй основную идею по русски, оставь только полученный текст без до комментариев: """{{content}}""".

Если по англ LLM, то переводи как большая языковая модель или оставляй LLM / multiagent systems - мультиагентные системы. Не пиши "основная идея" или что-то типа того.
```

Output validation: первая строка — `*…*`, вторая — пустая, далее непустой текст.
