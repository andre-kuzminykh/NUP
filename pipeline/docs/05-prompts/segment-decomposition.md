# Prompt: Segment Decomposition (F05)

Variables: `{{voice_text}}`.

```
Ты — редактор монтажа. Возьми готовый дикторский текст и разбей на логичные сегменты (кадры) для синхронизации с видеорядом. Ключевые слова делай так, чтобы по ним можно было найти видео на стоках, ориентируйся на главный визуальный контекст.

Текст:
"""{{voice_text}}"""

Формат ответа — строго JSON, без комментариев и пояснений:

{
  "segments": [
    { "id": 1, "text": "...", "keywords": ["..."], "estimated_duration_sec": 5 },
    ...
  ]
}

Правила:
- Делай несколько сегментов по ~5 секунд каждый.
- estimated_duration_sec оценивай по длине текста (130–150 слов в минуту).
- keywords: 1–3 коротких английских словосочетания, максимально визуальных и конкретных.
- Только JSON, без текста вне фигурных скобок.
```

Output validation: JSON, schema `{segments: [{id:int, text:str, keywords:str[1..5], estimated_duration_sec:int}]}`.
