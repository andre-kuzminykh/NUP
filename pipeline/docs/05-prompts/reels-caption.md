# Prompt: Reels Caption (F09)

Variables: `{{voice_text}}`.

```
Ты — профессиональный рилсмейкер. Напиши лучшее завлекательное описание для рилса на основе этого текста (без заголовка):

{{voice_text}}

В выводе только сам текст описания, без префикса «Описание:».
```

Output validation: 1–4 предложения, ≤500 символов.
